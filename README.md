# techbookfest7_vuepress_styleguide
本リポジトリは9/22(日)開催の技術書典7参加サークル「Gotanda Farm」の  
第3章「静的サイトジェネレーターVuePressでスタイルガイドを作ろう」の完成例です。

## 動作確認済み環境
- MacBook 12-inch Early 2015
- OS Mojave 10.14.5
- VuePress 1.0.3
- Nodejs 10.15.3
- yarn 1.17.3

## 各種パッケージインストール
```
yarn install
```

## ローカルサーバー起動
```
yarn start
```

## ビルド
```
yarn build
```
